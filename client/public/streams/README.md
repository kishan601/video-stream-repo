# HLS Streams Directory

This directory contains the HLS stream files generated by FFmpeg.

## Structure
- stream1/ - Stream 1 (Master, 0s delay)
- stream2/ - Stream 2 (0.3s delay)
- stream3/ - Stream 3 (0.6s delay)
- stream4/ - Stream 4 (0.9s delay)
- stream5/ - Stream 5 (1.2s delay)
- stream6/ - Stream 6 (1.5s delay)

Each stream directory contains:
- playlist.m3u8 - HLS playlist file
- segment_*.ts - Video segments

Files are automatically managed by FFmpeg (old segments are deleted).
